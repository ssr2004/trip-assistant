## 成都旅游攻略

### 城市特色

成都适合喜欢美食、慢节奏和人文生活的旅行者。城市节奏相对轻松，茶馆、火锅、小吃和熊猫元素都很有代表性。

### 必游景点

1. **成都大熊猫繁育研究基地**
   - 适合上午前往，熊猫活动更活跃
   - 亲子、情侣和拍照需求都比较适合

2. **宽窄巷子**
   - 适合感受成都老街和院落文化
   - 可安排小吃、茶馆和文创店体验

3. **武侯祠 + 锦里**
   - 适合人文历史爱好者
   - 锦里夜间氛围较好，适合轻松逛街

4. **春熙路 + 太古里**
   - 适合购物、拍照和城市休闲
   - 周边餐饮选择丰富

### 美食推荐

1. 火锅：适合能接受麻辣口味的用户。
2. 串串香：价格相对灵活，适合朋友出行。
3. 担担面、钟水饺、龙抄手：适合小吃打卡。
4. 盖碗茶：适合慢节奏休息。

### 三天行程

Day 1: 春熙路 → 太古里 → 宽窄巷子
Day 2: 熊猫基地 → 武侯祠 → 锦里
Day 3: 人民公园喝茶 → 文殊院 → 返程

### 注意事项

1. 成都热门餐厅排队较多，建议错峰用餐。
2. 不太能吃辣的用户可以提前说明微辣或鸳鸯锅。
3. 熊猫基地建议上午出发，避开午后高温和人流。
